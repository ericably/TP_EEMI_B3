# TP_EEMI_B3

Durée : 6 heures. 
À rendre : lien vers votre dépôt Git de l’étudiant

Contexte
Vous devez développer EventHub, une petite plateforme permettant à des étudiants de publier et de consulter des événements (soirée, atelier, conférence, tournoi…), classés par catégorie, avec espace membre et modérations.
Le projet reprend l'ensemble des notions vues en cours : structure d'un projet Symfony, contrôleurs et vues, Doctrine, formulaires et upload, sécurité et autorisations. Il est à réaliser en autonomie.

Bonus: Internationalisation, avec mise en place des traductions 

Conditions de réalisation
* Framework : Symfony 8.1 ou autre equivalent (PHP 8.4 ou autre équivalent).
* Base de données : Base de données au choix. Ce que vous préférez.
* Versioning : dépôt Git obligatoire, avec des commits réguliers (au minimum un commit par fonctionnalité majeure). On veut voir l'historique, pas un dépôt déposé d'un seul bloc.
* Autorisé : la documentation officielle de Symfony et vos propres notes de cours.
* Interdit : récupérer ou partager le code d'une autre personne.
